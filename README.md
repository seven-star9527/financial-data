# 泽坤财务工具中台 (Financial Tool Hub)

这是一个专为泽坤财务部门打造的、高度集成的自动化工具中台。它整合了多项日常财务核算、物流账单处理以及结算报表分析功能，旨在通过自动化流程提升办公效率。

## 🚀 快速启动

1.  **直接运行**：双击根目录下的 **`快速运行.bat`** 即可启动。
2.  **访问工具**：在浏览器中打开 [http://localhost:8081](http://localhost:8081)。

---

## 🛠 核心工具介绍

### 1. PDF 结算报表分析 (AI 驱动)
*   **功能**：深度解析 Amazon 原始 PDF 结算报告。
*   **特性**：
    *   集成 **DeepSeek AI**，自动识别多语言报表（如德语、法语、西班牙语）。
    *   自动提取：销售额 (Income)、各项费用 (Expenses)、广告费 (Ads) 以及税费明细。
    *   生成标准月度汇总 Excel 表，支持汇率自动换算（对接中国银行实时汇率）。

### 2. 国内运费账单处理
*   **功能**：快速匹配自建仓发货数据与 4PX 运费账单。
*   **特性**：
    *   支持 **多文件批量上传**。
    *   自动建立“运单号”索引，实现毫秒级数据匹配。
    *   输出结果直接填充至标准财务模板，保留原始样式。

### 3. 收款汇总表处理
*   **功能**：财务月度流水对账。
*   **特性**：
    *   整合 **万里汇 (WorldFirst)**、**PayPal**、**Shopify** 三方平台流水。
    *   自动核算各主体、各站点的收款情况。
    *   支持自定义汇率表进行统一折算。

### 4. 物流账单处理
*   **功能**：处理物流周/月度总账单。
*   **特性**：
    *   支持跨表汇总、费用分摊与店铺分组统计。
    *   提供详细的处理日志与异常数据提示。
